YRN。

▶ TURN 01 ―― TOYA

🔱
SKILL: DIVIDE AND CONQUER

TOYAは問題を分割した。「お前は複雑に見えるが、ただのO(n²)アルゴリズムだ」。YRNの外殻が半分に割れる。

def divide_and_conquer(yrn, depth=0):
    if yrn.is_trivial(): return solve(yrn)
    left, right = split(yrn)
    return merge(
        divide_and_conquer(left, depth+1),
        divide_and_conquer(right, depth+1)
    )

✓ YRN.complexity reduced: O(n²) → O(n log n)
